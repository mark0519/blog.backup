C语言学习笔记1.1

C字符串

c字符串基本知识

在C语言中，没有字符串变量，C语言保存字符串采用的是字符数组的形式。

C语言的字符串以null字符’\0’结束，例如字符串“HELLO”的定义：

char hello[6]={'H','E','L','L','O','\0'};

实际上，根据字符初始化规则，字符串hello只需要写出这样：

char hello[]={"HELLO"};

以下是定义的字符串内存表示:

Index 0 1 2 3 4 5
Variable H E L L O \0
Address 0x23451 0x23452 0x23453 0x23454 0x23455 0x23456

其实，不需要把 null 字符放在字符串常量的末尾。C 编译器会在初始化数组时，自动把 ‘\0’ 放在字符串的末尾，且在输出时null字符不可见。例如运行代码：

#include<stdio.h>
int main()
{
    char m[6]={'H','E','L','L','O','\0'};
    printf("%s\n",m);
    return 0;
}

编译之后，输出结果为：

HELLO

这里特别提到，当想完整输出某个字符数组m[8]时，可直接写为：

printf("%s",m); //省略m[8]中的[8]

其中%s代表以字符串形式输出，而C语言中也有%c表示单个字符的输出。

C语言中字符串输入

一般用：

char buf[100];
gets(buf);

或者

char buf[100];
scanf("%s",buf);

注意！在使用scnaf(“%s”,buf);时，输入的结尾一般为回车或者空格，所以scanf(“%s”,buf)没法在字符串中输入空格，而第一种使用gets()函数可以，但是，可以这样改进：

char buf[100];
scanf("%[^\n]",buf);

即可输入空格在字符串中。

具体原理如下：

%[]可以用来限制输入的内容，比如%[0-9]就表示只读入‘0’-‘9’之间的数字，%[a-z]表示只读入‘a’到‘z’之间的字母。我们这里的%[^\n]中的^就表示“非”的意思,\n就是我们平时所说的回车符，所以%[^\n]这个符号就是表示只读取 不是\n的字符。

但是用这种方式输入多组字符串时仍会出现问题，以为例如%[^\n]将会待接收到回车时停止，在数组buf中不含回车，但回车\n仍在缓冲区中，如果接下来仍有输入字符串，将会接收到这个\n，一般可以只用getchar()处理，如：

char buf1[100];
char buf2[100];

scanf("%[^\n]",buf1);
getchar();
scanf("%[^\n]",buf2);

C语言中操作字符串的部分函数（需要使用<string.h>头文件）

序号 函数 目的
1 strcpy(s1,s2); 复制字符串s2的内容到s1
2 strcat(s1,s2); 连接字符串s2到s1的结尾处
3 strlen(s1); 返回字符串s1的长度,不含null符
4 strcmp(s1,s2); 比较字符串s1和s2的大小
5 strchr(s1,ch); 返回一个指向s1中字符ch第一次出现的位置的指针,找不到返回NULL
6 strrchr(s1,ch); 返回一个指向s1中字符ch最后一次出现的位置的指针,找不到返回NULL
7 strstr(s1,s2); 返回一个指向s1中字符串s2第一次出现的位置的指针,找不到返回NULL

注意：strcmp(s1,s2)函数有两个参数，即要比较的两个字符串。strcmp()函数对两个字符串进行大小写敏感的(case-sensitiVe)和字典式的(lexicographic)比较，并返回下列值之一：

返回值 意义
<0 字符串s1小于字符串s2
0 字符串s1等于字符串s2
>0 字符串s1大于字符串s2

具体用法作用见实例2

下面是一些运用实例：

实例1（ strcpy()，strcat()，strlen() ）

#include<stdio.h>
#include<string.h>

int main()
{
	int i=0;
	char s1[]={"hello_"};
	char s2[]={"world"};
	char s3[]={"empty"};
	
	strcpy(s3,s2);           //strcpy()，复制字符串s2的内容到s1
	strcat(s1,s3);           //strcat()，连接字符串s3到s1的结尾处
	i=strlen(s1);            //strlen()，返回字符串s1的长度,不含null符
	printf("%d\n",i);
	printf("%s",s1);
	
	return 0;
 }

运行结果如下：

11
hello_world

实例2（ strcmp() ）

strcmp()
#include<stdio.h>
#include<string.h>

int main()
{
	int j=0;
	char s1[]={"ABC"};   
	char s2[]={"AbC"};
	
	printf("B = %d and b = %d\n",'B','b');    //输出B和b的ASCII值
	j = strcmp(s1,s2);
	printf("strcmp(ABC,AbC) = %d\n",j);       //比较
	
	return 0;
}

运行结果为：

B = 66 and b = 98
strcmp(ABC,AbC) = -1
strcmp()的一些变体
函数名 作用
strcmp() 对两个字符串进行大小写敏感的比较
strcmpi() 对两个字符串进行大小写不敏感的比较
stricmp() 同strcmpi()
strncmp(s1,s2,n) 把s1和s2进行比较，最多比较前 n 个字节(大小写敏感)
strnicmp(s1,s2,n) 把s1和s2进行比较，最多比较前 n 个字节(大小写不敏感)

如，如果将实例2中的strcmp(s1,s2)改成strmpi(s1,s2),则会返回 0

实例3（ strchr() ，strstr() ）

#include <stdio.h>
#include <string.h>

int main ()
{
   char m[]={"https://mark0519.github.io/"};
   char s1='.';
   char s2[]={"mark"};
   char *p1=NULL;
   char *p2=NULL;
   
   p1=strchr(m,s1);
   p2=strstr(m,s2);
   printf("The bytes after the '.' is %s\n",p1);     //输出结果会包含“.”
   printf("The bytes after the 'mark' is %s",p2);     
   
   return 0;
}

运行结果为：

The bytes after the '.' is .github.io/
The bytes after the 'mark' is mark0519.github.io/
