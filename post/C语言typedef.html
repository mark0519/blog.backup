C语言学习笔记1.5

Ctypedef

C 语言提供了 typedef 关键字，可以使用它来为类型取一个新的名字。

使用typedef关键字，为一个类型取一个新的名字，例如：

typedef long long int LLD;

这样定义了一个数据类型LLD,即为long long int 型的缩写，之后就可以这样写:

LLD a;

按照惯例，定义时会大写字母，以便提醒用户类型名称是一个象征性的缩写。但小写也是合法的。

也可以使用 typedef 来为用户自定义的数据类型取一个新的名字。例如，可以对结构体使用 typedef 来定义一个新的数据类型名字，然后使用这个新的数据类型来直接定义结构变量，如下：

#include<stdio.h>
int main()
{
	typedef struct data
	{
		char name[10];
		char sex[5];
		int age;
	}DATA;
	DATA mark0519;
	return 0;
 }

这等价于：

#include<stdio.h>
int main()
{
	struct data
	{
		char name[10];
		char sex[5];
		int age;
	};
	data mark0519;
	return 0;
 }
